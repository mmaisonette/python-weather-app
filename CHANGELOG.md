# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2023-12-20

### Added
- Flask endpoint (`app.py`) handling form submissions and error messaging.
- Weather module (`weather.py`) fetching coordinates and current conditions via OpenWeather.
- Bootstrap-based UI (`templates/index.html`) with responsive layout, tooltips, accessibility tweaks, and card-style weather output.
- pytest suite (`test_weather.py`) using monkeypatch to validate `weather.main`.
- Documentation: README usage/testing notes, MIT `LICENSE`, and `RELEASE_NOTES.md`.
